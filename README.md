The menu shows a GUI for selecting a calculator mode that calculates the required grade needed on the final exam with certain parameters. 

final_grade_calc
Allows user to enter multiple course grades and uses thresholds of 5 letter grades (A, B, C, D, F) to calculate the final grade needed for those letter grades. 
Returns information in the console and also saves it in a "req_grades.txt" file in a chart format. 

dropped_test
User is able to calculate required final grade needed for a course that has some number of tests dropped.
